How the scores are computed
  • LTM = Si/Sa.
  • STM = Sf/Sa.
  • WM = 0.5·STM + 0.5·(IQ/100).
  • AVP = 0.5·STM + 0.5·((tmax - t_access)/tmax).
  • FR = 0.5·STM + 0.5·avg(1/fa, 1/fh, 1/frlh).
  • PS = 0.5·STM + 0.5·((Tt - Ts)/Tt).
  • ATN = 0.5·STM + 0.5·( 0.5·(AVPI + PSI) blended with completion flag ).
  • CCS = weighted sum over all metrics.
